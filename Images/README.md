## 📌 Dashboard Features

The dashboard enables users to:

- Monitor Total Revenue, Orders, Customers, and Products.
- Analyze monthly sales trends over time.
- Compare revenue across product lines, countries, and cities.
- Identify the top 10 revenue-generating customers.
- Explore revenue distribution by deal size.
- Interact with slicers for Country, Product Line, Year, and Deal Size.
- View additional insights using report page tooltips.
